# NotebookLM 用プロンプト

以下のソース資料をNotebookLMにアップロードした上で、下記のプロンプトを使用してください。

---

## プロンプト

```
添付した資料をもとに、知り合いの経営者向けのプレゼン資料を作成してください。

### 資料の目的
- 新しく始めるキャバクラ求人紹介事業の全体像を説明する
- 「一緒にやりませんか」という提案ではなく、「こういう事業をやろうとしている」という事業説明
- 相手は経営者なので、ビジネスモデルと数字の根拠が伝わることが重要

### トーンと雰囲気
- ビジネス文書だけどカジュアル寄り（知り合い同士なので堅すぎない）
- 数字やファネルの具体性で信頼感を出す
- 押し売り感・営業感は一切なし。あくまで「こんなことやるんだけど」という説明

### スライド構成（12枚）
1. 表紙（事業名・日付）
2. ひとことで言うとこんな事業（サマリー）
3. ビジネスの流れ（広告→LINE→面談→入店→紹介料）
4. 求人のかけ方（店舗名義で出す仕組み）
5. 市場背景（なぜ成り立つのか）
6. 広告戦略（3媒体の使い分けと予算）
7. 集客ファネル（月間の数字の流れ）
8. 収益シミュレーション（悲観・標準・楽観の3パターン）
9. 収益モデルのバリエーション（買取/出勤バック/ハイブリッド）
10. 成長ステップ（3フェーズ）
11. リスクと対策
12. まとめ

### デザインの方向性
- シンプルで見やすいレイアウト
- 数字はテーブルやグラフで視覚的に
- ファネルの流れは図やフローチャートで表現
- 色使いは落ち着いたトーン（黒・紺・ゴールドなど）

### 注意点
- 「パートナー」「投資」「提案」などの言葉は使わない
- あくまで事業の説明資料として作ること
- 相手に何かを求める内容は入れない
```
